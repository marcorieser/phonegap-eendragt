phonegap-eendragt
====================

Installation
------------
npm install -g grunt-cli bower tsd

cd config

npm install

bower install

tsd reinstall

Commands
--------
# Development

grunt watch

http://localhost:63342/eendragt/build/

# Production

grunt

Plugins
-------------
# Remove
cordova plugin remove org.apache.cordova.statusbar

# Add
cordova plugin add org.apache.cordova.statusbar
