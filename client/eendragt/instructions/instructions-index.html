Spielregeln

In diesem Spiel kannst du gegen einen intelligenten Computer spielen.

Deine Schiffsflotte besteht aus einem 4-Master, zwei 3-Master sowie drei 2-Master Schiffe.

Die Schiffe werden im Zufallsprinzip auf das Spielfeld platziert. Dazu musst du nur auf den Button «Anordnen» drücken.
Wenn die Platzierung dir nicht zusagt, so drück einfach nochmals auf den Button, bis du mit der Platzierung einverstanden bist.

Um das Spiel zu starten drücke auf den Button «Bereit zur Schlacht».

Im Spiel hat jeder eine Kanonenkugel, welche er auf dem gegnerischen Spielfeld abfeuern kann. Wenn du einen Treffer gelandet hast, so hast du nochmals einen Schuss zur Verfügung, dies so lange bis du einen Leerwurf machst.

Wenn eine Bombe eines deiner Schiffe trifft, so siehst du sofort, welches Schiff getroffen wurde. Dies zeigt sich anhand zerstörter Segel oder Schiffsteile. Wenn eines deiner Schiffe ganz zerstört wird, siehst du nur noch die Überreste deines Schiffes als Wrackteil auf dem Spielfeld.

Und nun auf zur Seeschlacht!
