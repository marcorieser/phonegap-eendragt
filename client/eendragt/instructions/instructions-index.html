Spielregeln

Im Spiel kannst du gegen den Computer oder gegen einen Freund spielen. Wenn du gegen den Computer spielen willst, so drücke auf den Button «Kampagne». Danach gibt es eine Spielserie, welche über mehrere Spiele dauert. Pro Spiel wird das Level immer schwieriger. Die Schiffe werden im Zufallsprinzip auf das Spielfeld platziert. Dazu musst du nur auf den Button «platzieren» drücken. Wenn die Platzierung dir nicht zusagt, so drück einfach nochmals auf den Button. Nachdem die Schiffe platziert sind, beginnt jeder Spieler mit einer Kanonenkugel, welche er auf die gegnerischen Schiffe feuert. Wenn alle Felder eines Schiffes getroffen sind, sinkt es. Sind alle Schiffe eines Spielers versenkt, ist die Schlacht zu Ende. Derjenige Spieler gewinnt, der Schiffe oder noch Schiffsteile übrig hat. Und nun… auf zur Seeschlacht!
